Design Notes

Brand Consistency

All mockups use the existing brand palette (sage green, warm amber, charcoal navy) with Oswald headings and Open Sans body text.

Key Design Decisions

  • 430px max-width — targets iPhone 14 Pro Max as upper bound
  • Touch targets 44px+ — all buttons and controls meet accessibility minimums
  • Information density trade-offs — Option A variants favor richness, Option B variants favor speed/scanning
  • Sticky/fixed elements — cart summaries and CTAs are always reachable
  • Same data, different layouts — all mockups present the exact same data as desktop, reorganized for mobile

Navigation Approaches

  • Bottom tabs (Chef A, Owner A) — app-like, fast switching between sections
  • Hamburger + FAB (Chef B) — maximizes content area, action-first
  • Scroll-based (Owner B, Reports) — single-page dashboards, no nav distraction
